# Labs

Aqui ficam os laboratórios práticos do **security-foundations-lab**.

## Como usar
- Cada pasta em `labs/<tema>/` contém:
  - `README.md` com objetivo, contexto, passos e validação
  - `examples/` com exemplos, outputs e arquivos auxiliares
