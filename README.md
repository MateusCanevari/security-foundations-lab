# security-foundations-lab

Laboratório prático de fundamentos de segurança aplicados: autenticação/sessões, secrets, TLS/HTTPS, headers e CORS.

## Estrutura
- `docs/` teoria e guias curtos
- `labs/` exercícios práticos por tema
- `tools/` scripts e checklists
- `references/` links e cheatsheets

## Labs
- `auth-jwt/`
- `cookies-sessions/`
- `secrets-env/`
- `tls-https/`
- `headers-cors/`
